# utenkHome

Um site moderno e responsivo para apresentar utensílios de casa, com foco em modernidade, sofisticação e segurança para o lar.

## Funcionalidades

- **Página Inicial:** Layout atraente com imagem principal, texto motivacional, galeria de imagens com rolagem e seção "Antes e Depois".
- **Navegação por Cômodos:** Páginas dedicadas para Sala, Quarto, Cozinha e Banheiro, exibindo produtos específicos para cada ambiente.
- **Detalhes do Produto:** Cada produto possui texto chamativo, imagem, explicações detalhadas e link externo para compra.
- **Rodapé:** Informações sobre a autoria ("Feito por Andreia") e links para redes sociais (Instagram, Facebook, LinkedIn).

## Tecnologias Utilizadas

- **React:** Biblioteca JavaScript para construção da interface do usuário.
- **Vite:** Ferramenta de build para um desenvolvimento rápido e otimizado.
- **React Router:** Para gerenciar a navegação entre as diferentes páginas do site.
- **Bootstrap:** Framework CSS para design responsivo e componentes pré-estilizados.
- **Bootstrap Icons:** Biblioteca de ícones.

## Como Executar o Projeto Localmente

Siga os passos abaixo para ter o projeto rodando em sua máquina:

### 1. Navegue até o Diretório do Projeto

Abra o terminal ou prompt de comando e navegue até a pasta `meu-site-utensilios`:

```bash
cd meu-site-utensilios
```

### 2. Instale as Dependências

Instale todas as dependências do projeto usando npm:

```bash
npm install
```

### 3. Inicie o Servidor de Desenvolvimento

Para iniciar o servidor local e visualizar o site em seu navegador:

```bash
npm run dev
```

O site estará disponível em `http://localhost:5174/` (ou em uma porta similar, caso a 5174 esteja em uso).

### 4. Build de Produção

Para gerar uma versão otimizada do site para deploy:

```bash
npm run build
```

Os arquivos de produção serão gerados na pasta `dist/`.
